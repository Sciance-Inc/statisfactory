class statisfactory.IO.artifacts.artifact_interactor.ArtifactInteractor(artifact, *args, session: statisfactory.session.BaseSession, **kwargs)¶

Bases: statisfactory.logger.MixinLogable, MixinInterpolable

Describe the Interactor’s interface. An interactor wraps the loading and saving operations of a Artifact. The user can implements custom interactors. To do so, the user should implements the interface desbribes in this class. An artifact and a Session object are available when the artifact is called by the Catalog.

Validation of the Extra parameter is possible through the definition of an inner class named Extra. The inner class schould be a pydantic dataclasse or a pydantic base model to allow for automatic validation.
