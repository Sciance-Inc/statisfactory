class statisfactory.IO.ArtefactInteractor(artefact, *args, session: statisfactory.session.Session, **kwargs)¶

Bases: statisfactory.logger.MixinLogable, MixinInterpolable

Describe the Interactor’s interface. An interactor wraps the loading and saving operations of a Artefact. The user can implements custom interactors. To do so, the user should implements the interface desbribes in this class. An artefact and a Session object are available when the artefact is called by the Catalog.
